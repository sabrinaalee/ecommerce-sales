
# E-commerce Sales Analysis (Python + SQL + Tableau)

## Overview
This project analyzes over 500,000 retail transactions to uncover key insights into sales performance, customer behavior, and product trends.

The analysis combines Python for data cleaning and exploration, SQL for querying structured insights, and Tableau for interactive dashboard visualization.

---

## Tools Used
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Tableau
- Google Colab

---

## Dataset
Online Retail transactional dataset containing:
- Invoice-level sales data
- Product information
- Customer IDs
- Country-level data

---

## Key Business Questions
- What are the overall sales trends over time?
- Which products generate the most revenue?
- Who are the top customers?
- Which countries contribute the most revenue?
- How concentrated is revenue among customers/products?

---

## Key Insights
- Revenue is highly concentrated among top customers and products.
- Sales show clear monthly variation and seasonality.
- A small group of customers drives a large portion of total revenue.
- The United Kingdom dominates overall sales activity.
